Projelerim

Picker 3D Clone

Hyper Casual Game

Bu proje, Picker3D oyununa benzer bir oyun olarak geliştirilmiştir. Kod yapısı, SOLID prensiplerine uygun şekilde tasarlanmış ve yazılımın esnekliğini artırmak için çeşitli design pattern'ler kullanılmıştır. Projede Observer Pattern ile olay yönetimi ve güncellemeler sağlanırken, Singleton Pattern merkezi yönetim yapısını oluşturmuştur. Command Pattern komut tabanlı işlemler için kullanılmış, Factory Pattern ile dinamik nesne oluşturma işlemleri yönetilmiş ve Actor Pattern sayesinde oyun nesnelerinin bağımsız davranışları modellenmiştir. Ayrıca, oyun verileri Scriptable Object yapıları ile ayrıştırılarak daha modüler bir yapı oluşturulmuş ve sahne içi prefab yönetimi, signal sistemi ile optimize edilmiştir.

Unity 3D DesignPatterns Android

Personel Kayıt Sistemi

Web Geliştirme

Bu proje, ASP.NET Core MVC ve .NET 8.0 kullanarak geliştirdiğim bir personel yönetim sistemidir. Verileri Microsoft SQL Server üzerinde saklıyor ve Entity Framework Core ile yönetiyorum. Güvenlik tarafında HTTPS, HSTS ve kullanıcı yetkilendirme sistemlerini entegre ettim. MVC yapısını kullanarak modüler ve genişletilebilir bir mimari oluşturdum. Arayüzü Razor View Engine ile geliştirdim ve mobil uyumlu olmasına özen gösterdim. Clean Code ve SOLID prensiplerine bağlı kalarak, hem okunaklı hem de sürdürülebilir bir yapı oluşturmaya çalıştım. Kısacası, güvenli, ölçeklenebilir ve kullanıcı dostu bir sistem ortaya çıkardım.
